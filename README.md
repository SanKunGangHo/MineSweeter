# MineSweeter

> 2019년 대학교 3학년 학교 과제로 제작한 3D 지뢰찾기 게임입니다.

## 소개

고전 지뢰찾기를 3D 환경에서 재해석한 게임입니다.
플레이어 캐릭터를 직접 조작해 지뢰가 숨겨진 필드를 탐색하고, 모든 지뢰를 깃발로 표시하면 승리합니다.

- 제작 시기: 2019년
- 제작 목적: 대학교 수업 과제
- 플랫폼: Android / WebGL

## 조작 방법

| 플랫폼 | 이동 | 발굴 / 깃발 |
|--------|------|-------------|
| PC / WebGL | WASD 또는 방향키 | 화면의 버튼 클릭 |
| 모바일 | 조이스틱 | 화면의 버튼 터치 |

## 기술 스택

- Unity 2019
- C#
- Android Build / WebGL Build

## 게임 방법

1. 화면을 클릭하면 게임이 시작됩니다.
2. 캐릭터를 블록 위로 이동시킵니다.
3. 블록에 올라서면 **Dig(발굴)** 또는 **Flag(깃발)** 버튼이 나타납니다.
4. 지뢰가 없는 블록을 발굴하면 주변 지뢰 개수가 표시됩니다.
5. 지뢰 블록을 발굴하면 게임 오버입니다.
6. 모든 지뢰 위치에 깃발을 꽂으면 승리합니다.
